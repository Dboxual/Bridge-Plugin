Required jars for compilation — place them in this folder:

  paper-api.jar         Copy from ../Pinpoint/libs/paper-api.jar
  fawe-bukkit.jar       Download FastAsyncWorldEdit for Paper 1.21.x from:
                        https://modrinth.com/plugin/fastasyncworldedit
                        Rename the downloaded jar to fawe-bukkit.jar

  adventure-api.jar     Copy from ../Pinpoint/libs/adventure-api.jar
  adventure-key.jar     Copy from ../Pinpoint/libs/adventure-key.jar
  examination-api.jar   Copy from ../Pinpoint/libs/examination-api.jar
  guava.jar             Copy from ../Pinpoint/libs/guava.jar
  bungeecord-chat.jar   Copy from ../Pinpoint/libs/bungeecord-chat.jar
  jetbrains-annotations.jar  Copy from ../Pinpoint/libs/jetbrains-annotations.jar

Quick copy command (run from the TheBridge directory):
  cp ../Pinpoint/libs/*.jar libs/
  # Then download fawe-bukkit.jar manually
